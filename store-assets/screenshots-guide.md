# Screenshot Guide untuk Store Submission

## Screenshot yang Diperlukan

### 1. **Popup Extension** (1280x800 atau 640x400)
- Screenshot popup extension saat diklik
- Tunjukkan statistik blocking
- Tampilkan tombol toggle dan dashboard

### 2. **Dashboard Overview** (1280x800)
- Screenshot halaman dashboard utama
- Tunjukkan statistik total blocked
- Tampilkan navigasi menu

### 3. **Filter Lists** (1280x800)
- Screenshot halaman filter lists
- Tunjukkan daftar filter yang tersedia
- Tampilkan checkbox untuk enable/disable

### 4. **Whitelist Management** (1280x800)
- Screenshot halaman whitelist
- Tunjukkan cara menambah domain
- Tampilkan daftar domain yang di-whitelist

### 5. **Settings Page** (1280x800)
- Screenshot halaman pengaturan
- Tunjukkan berbagai opsi konfigurasi
- Tampilkan toggle untuk berbagai fitur

## Tips Screenshot:
1. Gunakan browser dengan tampilan bersih
2. Pastikan extension sudah memblokir beberapa iklan untuk statistik
3. Gunakan resolusi tinggi untuk kualitas terbaik
4. Tambahkan beberapa domain ke whitelist untuk demo
5. Pastikan semua teks terlihat jelas

## Icon Requirements:
- **128x128px** - Icon utama untuk store
- **48x48px** - Icon medium
- **16x16px** - Icon kecil

Icon sudah tersedia di `src/img/icon.svg` dan dapat di-convert ke PNG dengan ukuran yang diperlukan.

## Promotional Images (Opsional):
- **1400x560px** - Large promotional image
- **920x680px** - Small promotional image

Buat gambar promosi yang menunjukkan:
- Logo Vaizbtgads
- Tagline: "Content Blocker untuk Browser"
- Fitur utama dalam bullet points
- Screenshot mini dari interface
